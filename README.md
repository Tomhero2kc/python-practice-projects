# Python Practice Projects

This repository contains small Python projects I am building as I learn and practise the basics.

I am using these projects to improve my understanding of functions, files, JSON, APIs and error handling.

## Projects Completed

### Simple Calculator

This was my first project. It is a command-line calculator that can add, subtract, multiply and divide numbers.

I also added basic error handling for invalid entries and division by zero.

View the project here: [`calculator/`](calculator/)

### CSV File Reader

I built this project to practise reading data from CSV files.

The program reads a small file containing names, ages and cities, then displays each row in the terminal. I also added error handling for missing files and incorrect column names.

View the project here: [`csv-reader/`](csv-reader/)

### GitHub Profile Fetcher

I built this project to practise working with APIs in Python.

The program asks for a GitHub username, gets public profile information from the GitHub API and displays it in the terminal.

While building it, I practised sending requests, reading JSON data and handling connection errors.

View the project here: [`api-fetcher/`](api-fetcher/)
