# Simple Contact List

I created this small contact list while practising how to work with JSON files in Python.

The program reads saved contacts from a JSON file and displays them in the terminal. It also allows the user to add a new contact and saves the information for the next time the program runs.

## What the Program Does

- Loads contacts from a JSON file
- Displays saved names, email addresses and phone numbers
- Allows the user to add a new contact
- Saves new contacts automatically
- Handles missing or invalid JSON files

## Project Files

- `json_practice.py` contains the Python program
- `contacts.json` stores the contact information

## How to Run It

From the main repository folder, run:

```bash
python json-practice/json_practice.py
```

## What I Learned

While working on this project, I practised:

- Reading information from a JSON file
- Turning JSON data into Python lists and dictionaries
- Adding new information to a list
- Saving updated information back to a file
- Handling missing files and invalid JSON
